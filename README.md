# TECHOLUTION WEBSITE

### Development

**Dependencies**

- [Nodejs](https://nodejs.org/en/download/)
- [Gatsby Cli](https://www.gatsbyjs.com/docs/tutorial/part-0/#gatsby-cli)
  **Installation**

Run the following commands in the terminal(mac os)/command prompt(windows).

- git clone [Github Link](https://github.com/Techolution/techo-website.git)
- cd techo-webiste
- npm install

## To Run App

- npm run develop

### To Build Individual Apps

- npm run build
